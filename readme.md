## 4. 프로젝트 이름 변경

![4](/img/4.png)

Crtl + F를 누른 후

①을 눌려 창을 확장하고

②칸에 App1 입력

③칸에 현재 프로젝트 명 입력

탐색 범위는 전체 솔루션으로 선택하고

④를 눌려 모두 바꾸기 (Alt + A)를 해줍니다.
